# Отчёт по домашнему заданию HW06

## 1. Общая информация

**Выбранный датасет:** S06-hw-dataset-01.csv  
**Тип задачи:** Бинарная классификация  

## 2. Описание данных

### 2.1 Основные характеристики

- **Количество строк:** 1000
- **Количество признаков:** 10
- **Целевая переменная:** `target`
- **Тип задачи:** Бинарная классификация
- **Баланс классов:** Имеется дисбаланс (91% класса 0, 9% класса 1)

### 2.2 Распределение классов

**Доли классов:**
- Класс 0: 90.9%
- Класс 1: 9.1%

### 2.3 Предобработка данных

- Проверка на пропуски: пропусков не обнаружено
- Категориальные признаки отсутствуют
- Все признаки являются числовыми

## 3. Методология эксперимента

### 3.1 Разделение данных

- **Метод:** `train_test_split` из sklearn
- **Размер тестовой выборки:** 25%
- **Стратификация:** Да (по целевому признаку)
- **Random state:** 42
- **Размер train:** 750 строк
- **Размер test:** 250 строк

### 3.2 Метрики качества

**Основные метрики:**
1. **Accuracy** - общая точность классификации
2. **F1-score** - гармоническое среднее precision и recall (binary для бинарной задачи)
3. **ROC-AUC** - площадь под ROC-кривой

### 3.3 Модели

**Baseline модели:**
1. Dummy Classifier (strategy='most_frequent')
2. Logistic Regression с StandardScaler

**Основные модели:**
1. Decision Tree Classifier
2. Random Forest Classifier  
3. Gradient Boosting Classifier

### 3.4 Подбор гиперпараметров

- **Метод:** GridSearchCV
- **Количество фолдов:** 5 для Decision Tree, 3 для остальных
- **Scoring:** F1-score (binary)
- **Особенность:** Использован n_jobs=1 для избежания ошибок многопроцессорности
